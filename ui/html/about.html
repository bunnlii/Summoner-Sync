About Summoner Sync

Summoner Sync helps you compare up to five League of Legends players by pulling recent match stats and champion mastery data straight from Riot’s APIs. It presents clean insights so you can quickly understand performance and style.

Player Insights

See averages like KDA, CS/min, vision, damage, and lane tendencies computed from recent matches.

Side-by-side Compare

Add multiple Riot IDs (GameName#Tag) to build a quick comparison board for a lobby or team.

Champion Mastery

View each player’s top champion masteries with names, roles, and points at a glance.

How it works

  • Enter a Riot ID like GameName#TAG and hit Search.
  • Our backend fetches the player’s PUUID, recent matches, and rankings using Riot APIs.
  • We compute readable summaries (averages and highlights) and render your comparison.
